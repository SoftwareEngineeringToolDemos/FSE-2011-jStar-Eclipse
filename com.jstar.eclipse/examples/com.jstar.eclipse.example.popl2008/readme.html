Popl2008 Example

This example contains three source files Cell.java, Recell.java, and DCell.java with specifications included in them using java annotations.

The jStar folder contains logic and abstraction rules of the source files. It also contains specification, logic rules and abstraction rules for an external class java.lang.Object.

To verify one of these source file, select it in the Package Explorer. Open its context menu, select jStar Verification and Verify.
